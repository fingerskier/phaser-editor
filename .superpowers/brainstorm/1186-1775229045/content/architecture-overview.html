Key Insight

The Command System is the heart of the architecture. Every mutation — whether from a user clicking in the UI or Claude calling an MCP tool — flows through the same command pipeline: validate → execute → push to undo stack → update Svelte stores → persist to disk. This guarantees consistent undo/redo regardless of mutation source.
